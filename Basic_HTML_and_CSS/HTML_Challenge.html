The Article Tag

The article tag specifies independent, self-contained content. You can use it for blogs or news posts.

The Detail Tag

The detail tag is used to provide more information on a certain topic. It is almost like a drop down, when you click on it it shows further information, and if you click on it again it will close that information.

The Time Tag and Datetime Attribute

The time tag defines a specific time. The datetime attribute is used to translate time into a machine readable time, which leads the browser to offer date reminders for the user.
